# SnipTool Feature Overview

This document summarizes current end-user functionality from the codebase.

## Capture Modes
- Region capture via overlay selection
- Active window capture
- Fullscreen capture
- Repeat last region

## Video Recording
- Region video recording
- Window video recording
- Fullscreen video recording
- Start, pause/resume, stop controls
- Optional audio capture

## Workflow
- Global hotkeys trigger capture directly
- Auto-save to disk on each capture
- Optional copy-to-clipboard after save
- Toast actions after save:
  - Undo last capture
  - Open capture folder
  - Open editor for the latest capture

## Burst Sessions
- Start a burst to group captures in one dated session folder
- End burst to save directly to root folder
- Burst HUD status while active

## Library and Editing
- Built-in library window for browsing captures
- Image editor window for quick markups and edits

## Settings
- Save path and filename template
- Customizable hotkeys
- Dark/light theme
- Video default audio toggle
